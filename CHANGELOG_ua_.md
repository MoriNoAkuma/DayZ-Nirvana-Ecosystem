# v1.0.0 Повний реліз
## Реалізовано

- !!Інтеграція DayZ мода з системою подій (join/leave/kill/game_time через вебхук)
- Розумна каскадна система отримання статусу сервера (DayZ → CFTools → BattleMetrics → кеш)
- Автоматична валідація свіжості даних гри (heartbeat TTL = 180 сек для DayZ мода)
- Оптимізація парсингу PendingStore з перевіркою часових міток оновлень
- Розділено логіку Discord-бота та веб-сайту
- Інтегровано Jinja2 для веб-сайту
- Перенесено інформаційну панель у .json для спрощення оновлень
- Додано fallback для коректного відображення даних сервера (DayZ Server → CFTools → BattleMetrics)
- Оновлено систему Discord-бота: додано можливість гнучкіше керувати даними через адмін-панель на сайті

## Гарячі виправлення

- Виправлено неправильний підрахунок онлайну гравців (передчасне виключення гравця із сервера)
- Виправлено: Discord бот тепер коректно відображає час на сервері (замість N/A)
- Виправлено некоректну роботу системи ліцензування програми
