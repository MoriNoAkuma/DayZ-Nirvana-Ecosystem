# v1.0.0-rc.3
## Реалізовано
- !!Реалізація динамічного конструктора тікетів Discord через веб-панель (до 5 кастомних полів)
- Розумна синхронізація пов'язаних промокодів (автоматичне оновлення нагород на DayZ сервері при зміні шаблону)
- Приватна (ephemeral) система оцінювання для закритих тікетів для запобігання накрутці адміністраторами
- Динамічне "гаряче" перезавантаження (hot-reloading) шляху до LIS_Promocodes (без необхідності перезапускати бота)
- Додано сувору валідацію SteamID (лише цифри) для форм тікетів
- Покращення UI/UX веб-панелі (візуальний конструктор тікетів, ліміти для форм) та оновлена документація

## Гарячі виправлення
- Виправлено: помилка атрибута DonationConfig під час збереження налаштувань з веб-конструктора
- Виправлено: втрата цілісності метаданих (ProductID та DisplayName) під час видачі товарів або відкриття кейсів
